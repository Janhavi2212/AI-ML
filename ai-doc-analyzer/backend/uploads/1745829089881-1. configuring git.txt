1. configuring git
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

2.Initializes a new Git repository in your current directory.
git init 

3. ACP
when files change, to be added
git add filename.txt      # Stage a specific file
git add .                       # Stage all changes

git commit -m "Your commit message"

4. Displaying status.
git status

5. shows commit history
git log

6.Linking Local Repository to GitHub:
Navigate to your local project directory in the terminal.
Use the command:
git remote add origin <repository-url>
Replace <repository-url> with the URL of your GitHub repository.
Pushing Local Changes:
Stage your changes with:
git add <file>
Commit the changes with:
git commit -m "Your commit message"
Push the changes to GitHub using:
git push -u origin main
